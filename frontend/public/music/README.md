# Bundled Music Tracks

Place three short loopable MP3 files here:

- `track1.mp3` — "Happy Vibes"
- `track2.mp3` — "Party Time"
- `track3.mp3` — "Celebration"

These are referenced by the game's music player. If the files are missing,
the music selector will still work but no audio will play for bundled tracks.

You can use royalty-free tracks from sources like:
- https://pixabay.com/music/
- https://freemusicarchive.org/
- https://incompetech.com/
